sessionStorage

DIe Daten werden nur bis zum Beenden der Sitzung(Browser wird geschlossen) gespeichert Danach verfallen sie.

Die Daten werden nie zum Server transferiert.

DIe Maximale Größe leigt bei 5Mb

localStorage

FUnktioniert identisch wie die sessionStorage, aber die Daten werden auch nach Schleißen des Browsers beibehalten. Die Daten werden erst gelöscht, wenn der Benutzer seinen Cache leert, oder wir sie per JavaScript löschen.

Aufbau mit dem von Cookies vergleichbar: Schlüssel = Wert;

Aufrufen: sessionStorage.SchlüsselName => Gibt Wert des Schlüssels zurück localStorage.SchlüsselName => GIbt Wert zurück localStorage["Schlüsselname"] local/sessionStorage.getItem(Schlüssel) => Wert wird zurückgegeben

Speichern eines neuen Schlüssel-Wert Paares: local/sessionStorage.setItem("Schlüssel", "Wert")

Eigenschaft: local/sessionStorage.length => Gibt zurück wie viele Schlüssel-Wert-Paare vorhanden sind.

Methoden: ENtfernen eines S-W-Paares: session/localStorage.removeItem("Schlüssel") Alle Daten löschen: session/localStorage.clear() Schlüssel ermitteln: session/localStorage.key(Zahl) => GIbt uns den Schlüssel für die jeweilige Index-Position zurück.
